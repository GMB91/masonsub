Design Philosophy
Color Palette:
Primary Colors:
- Indigo/Blue: #4F46E5 (buttons, links, active states)
- Slate: #64748B (text, borders)
- White: #FFFFFF (cards, backgrounds)
- Gray-50: #F9FAFB (page backgrounds)

Status Colors:
- Green: #10B981 (success, completed, claimed)
- Blue: #3B82F6 (info, in progress, researching)
- Yellow: #F59E0B (warning, pending, contacted)
- Red: #EF4444 (error, failed, deep search required)
- Purple: #8B5CF6 (contractor, specialty)

Gradients:
- Primary: from-blue-500 to-indigo-600
- Success: from-green-400 to-emerald-500
- Background: from-slate-50 to-blue-50

- H1: text-3xl font-bold (30px)

Body:
- Base: text-base (16px)
- Small: text-sm (14px)

Font Family: System UI (Tailwind default)
Spacing System:
Padding/Margin Scale:
- xs: 0.5rem (8px)
- sm: 0.75rem (12px)
- base: 1rem (16px)

- sm: 640px
